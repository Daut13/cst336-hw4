Step 1: Download Android Studio

Click the link here to download Android Studio!

Step 2: Open a New Project

Once you have downloaded Android Studio, open it up and select to start a new android project. You can create an Empty Activity.
Make sure to leave the configuration alone, and to ensure Java is your selected language to code. Then click finish!

Step 3: Understanding Your Files

The main files you will be using are the following:

  • app > java > com.example.myfirstapp > <%= randomFileName %>
  • app > res > layout > activity_main.xml
  • app > manifests > AndroidManifest.xml
  • Gradle Scripts > build.gradle

Once you open all of these files, we can finally start with making your app!
