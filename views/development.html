Step 4: Opening Layout Editor

One of the best parts of Android Studio and it's development, is that you can use the layout editor to create your application. Open up <%= randomFileName %>, and click on the tab labeled design.

Step 5: Using Layout Editor

Once you have opened up the design page, you should see a menu with a bunch of different panels you can add over to your application. All you have to do is click and drag your preferred item over to your application and place it!

Step 6: Make a Design

This is probably the easiest way for you to start with a design, without even having to start to code anything from here. You can begin simply by placing things where you would like them, and once you've finalized your design, you can get started straight onto the coding side of things. If you are familiar with Java, most of the code should be pretty straight-forward.

To keep this guide simple, I will not be going over the code for your application, but you can create whatever you'd like! If you'd like help with creating something, please follow the guide here. Once you've got something you are happy with, you can move to the Finish page to learn how to build and run your app!
